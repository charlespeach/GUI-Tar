GUI Tar Help

How to use Extractor

Standard Use

Launch GUI Tar, click on the Extract tab, and then select the file you wish to uncompress or unarchive, and click on the Extract button. For some files, such as .tar, .tgz, .rar, or .zip, they can be placed in a user-specified location by setting the Directory field.

Drag and Drop

Drag and drop a compressed or archived file onto the GUI Tar application icon, and GUI Tar will launch automatically and uncompress the file.

If the collection of files contains any compressed or achived files, GUI Tar will ask whether to compress or extract the files. Select Extract, and the first compressed or archived file will be extracted.
